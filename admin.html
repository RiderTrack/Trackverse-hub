❓ Cómo usar este panel

📦 Agregar una app nueva

  1. Andá al tab Apps
  2. Tocá "➕ Agregar app"
  3. Completá: nombre, icono (emoji), color, repo (ej: rudyalen/fitwallet)
  4. Tocá "Guardar"
  5. La app aparece en el catálogo inmediatamente

🎨 Cambiar colores

  1. Andá al tab Theme Studio
  2. Elegí los colores con el selector de color
  3. Tocá "Guardar tema"

🔑 Configurar GitHub Token

  1. Entrá a github.com/settings/tokens
  2. Generate new token (classic)
  3. Nombre: "TrackVerse Hub"
  4. Marká el permiso repo
  5. Generá y copiá el token
  6. Pegalo acá en tab Config

📱 Cómo se suben APKs al catálogo

Cada vez que hacés git push al repo de una app (ridertrack, clientetrack, etc.), el workflow de GitHub Actions construye el APK y crea un Release automáticamente. El catálogo lee los releases vía API y los muestra. No tenés que subir nada manualmente.

⚠️ Importante

  • Los cambios se guardan en apps.json del repo trackverse-hub
  • Para que los cambios funcionen, necesitás el GitHub Token configurado
  • Si perdés el token, podés generar uno nuevo y reemplazarlo
  • El catálogo se actualiza cada vez que entrás a la URL (no hay cache)
